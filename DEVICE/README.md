# 📱 DEVICE模块

DEVICE模块是EasyAIoT系统的核心组件之一，提供IoT设备管理、产品管理、数据标注、规则引擎等完整的设备管理功能。

## 功能特性

### 📱 设备管理
- **设备CRUD**: 支持设备的新增、修改、删除、查询等基础操作
- **设备状态监控**: 实时监控设备在线/离线状态，支持设备连接状态查询
- **设备信息管理**: 支持设备基础信息维护，包括设备标识、名称、协议类型等
- **子设备管理**: 支持边设备下的子设备管理，支持设备拓扑关系管理
- **设备影子**: 支持设备影子功能，实现设备状态同步

### 📦 产品管理
- **产品CRUD**: 支持产品的新增、修改、删除、查询等基础操作
- **物模型管理**: 支持产品物模型定义，包括属性、事件、服务等
- **产品信息维护**: 支持产品基础信息维护，包括产品标识、名称、描述等

### 🏷️ 数据标注
- **数据集管理**: 支持数据集列表展示和基础信息管理
- **标签管理**: 支持数据集标签的创建、编辑和管理
- **图片数据集**: 支持图片数据集的导入、处理和标注
- **视频数据集**: 支持视频数据集的导入、处理和标注
- **视频流帧捕获**: 支持从视频流中捕获关键帧用于标注
- **标注工具**: 提供图像数据集标注工具，支持目标检测标注

### ⚙️ 规则引擎
- **规则管理**: 支持规则的查询、创建、修改和删除
- **MQTT节点**: 支持MQTT节点配置，实现设备数据采集
- **Kafka节点**: 支持Kafka节点配置，实现数据流处理
- **函数管理**: 支持自定义函数管理，实现数据转换和处理
- **JS引擎**: 支持JavaScript引擎，实现灵活的数据处理逻辑

### 🤖 算法商店
- **人脸识别**: 提供人脸识别算法服务
- **行人跟踪**: 支持跨摄像头行人跟踪功能
- **车辆识别**: 支持车辆特征识别功能
- **车牌识别**: 提供车牌识别算法服务

### 🏗️ 基础设施
- **定时任务**: 支持定时任务的管理和调度
- **系统配置**: 支持系统配置参数的管理
- **监控中心**: 提供系统监控中心，实时监控系统运行状态
- **文档管理**: 支持文档管理系统，实现文档的上传、下载和管理
- **数据库监控**: 支持MySQL数据库性能监控
- **缓存监控**: 支持Redis缓存监控
- **应用监控**: 支持Java应用性能监控
- **链路追踪**: 支持分布式链路追踪监控
- **访问监控**: 支持文档访问监控和统计

### 👥 系统管理
- **租户管理**: 支持多租户管理，实现租户隔离
- **用户管理**: 支持用户的创建、修改、删除和查询
- **角色权限**: 支持基于角色的访问控制（RBAC），实现细粒度权限管理
- **菜单管理**: 支持系统菜单的动态配置和管理
- **部门组织**: 支持部门组织架构管理
- **岗位管理**: 支持岗位信息管理
- **字典管理**: 支持系统字典数据管理
- **客户端管理**: 支持IoT客户端的管理和配置
- **第三方登录**: 支持第三方登录集成（如OAuth、SSO等）
- **地区管理**: 支持地区信息管理
- **数据源配置**: 支持多数据源配置管理
- **代码生成**: 提供代码生成工具，快速生成业务代码
- **系统日志**: 支持系统操作日志的记录和查询
