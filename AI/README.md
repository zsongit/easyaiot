# 🤖 AI模块

AI模块一个集成了AI模型训练、部署、管理和服务化的综合平台。该平台支持YOLOv8目标检测模型的完整生命周期管理，包括训练、导出和部署为在线服务。

## 功能特性

- **模型训练**: 支持YOLOv8/YOLO11n目标检测模型训练，完整的训练任务管理和进度跟踪
- **模型导出**: 支持将训练好的模型导出为多种格式（ONNX、TorchScript、TensorRT、OpenVINO等）
- **模型推理**: 支持图片、视频、RTSP流的实时推理，提供推理任务管理和结果存储
- **模型部署**: 支持将模型部署为独立的RESTful API服务，支持多实例部署和负载均衡
- **服务管理**: 提供模型服务的启动、停止、状态查询、心跳监控等功能
- **LLM服务**: 支持大语言模型（LLM）配置和管理，支持多厂商API（OpenAI、Anthropic等），支持文本、视觉、多模态分析
- **OCR服务**: 支持光学字符识别（OCR）功能，可识别图片中的文字信息
- **语音服务**: 支持语音识别和语音合成功能
- **集群推理**: 支持分布式集群推理，实现高并发推理请求处理
- **日志跟踪**: 完整的训练过程日志记录和查询，支持推理任务日志追踪
- **MinIO集成**: 支持从MinIO对象存储中读取数据集和模型文件，支持推理结果存储
- **Nacos集成**: 支持服务注册与发现，实现服务自动注册和健康检查
