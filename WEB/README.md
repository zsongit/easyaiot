# EasyAIoT WEB模块
EasyAIoT WEB 是一个基于现代前端技术构建的前端项目，提供设备管理、模型训练、数据监控、用户权限管理等功能。

## 功能特性

- **设备管理**: 支持IoT设备的添加、删除、修改、查询，设备状态实时监控，设备分组和目录管理
- **摄像头管理**: 支持摄像头设备的配置和管理，支持ONVIF协议自动发现，支持视频流预览和播放
- **视频监控**: 支持多路视频流实时预览，支持视频卡片和列表两种展示模式，支持视频流控制
- **抓拍管理**: 支持抓拍空间配置，支持抓拍任务管理（实时抓拍、定时抓拍），支持抓拍图片查看和管理
- **录像管理**: 支持录像空间配置，支持录像文件查询和回放，支持录像下载
- **算法任务**: 支持算法任务配置和管理，支持实时任务和定时任务，支持任务状态监控
- **AI模型训练**: 支持在线模型训练与参数调优，支持训练任务管理和进度查看，支持模型管理
- **模型推理**: 支持图片、视频推理任务提交，支持推理结果查看和下载
- **数据可视化**: 实时展示设备数据，支持多种图表分析（折线图、柱状图、饼图等）
- **告警管理**: 支持告警信息查看和管理，支持告警规则配置
- **权限控制**: 基于角色的访问控制（RBAC），支持用户、角色、菜单权限管理
- **系统管理**: 支持租户管理、用户管理、部门管理、字典管理等系统基础功能
- **响应式设计**: 支持多端访问（PC、平板、手机），自适应不同屏幕尺寸
- **国际化支持**: 多语言切换，支持中英文界面

## 技术栈

- **框架**: Vue 3
- **语言**: TypeScript
- **构建工具**: Vite
- **UI框架**: Element Plus / Ant Design Vue
- **状态管理**: Pinia
- **路由**: Vue Router
- **HTTP客户端**: Axios
- **代码规范**: ESLint + Prettier
